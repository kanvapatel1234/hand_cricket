# 🏏 Hand Cricket Game

A simple Hand Cricket game built using **HTML, CSS, and JavaScript**. Play against the computer by choosing runs (1, 2, 4, or 6). The computer randomly selects its move, and the match continues until either side gets out.

## Features

* 🎮 Simple and interactive gameplay
* 🤖 Random computer moves
* 🏏 Batting and Bowling modes
* 📊 Live score updates
* 🏆 Win, Lose, and Tie detection
* 🔄 Play Again button to restart the match
* 🎨 Modern responsive UI

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Play

### Batting

1. Click any run (1, 2, 4, or 6).
2. The computer randomly chooses a number.
3. If both numbers are different, your selected runs are added to your score.
4. If both numbers are the same, you are **OUT**.

### Bowling

1. After getting out, start bowling.
2. Choose a number (1, 2, 4, or 6).
3. The computer bats by selecting a random number.
4. If the numbers match, the computer is **OUT**.
5. If the computer reaches your target before getting out, it wins.
